**Local Network Chat Server**

**Required packages**
```
$pip install flask
$pip install flask-socketio
$sudo apt-get install python3
```

```
Note: Make sure that the port you choose is allowed in your firewall
Warning: There is no SQL server to save files to, so all messages will be
         lost if the server is shut down for any reason.
```
